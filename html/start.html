Getting started


Install the command line tool.

npm install bauer-cli -g

Create a new bauer script.

bauer init script

Create a new bauer plugin.

bauer init plugin

Adds plugins to the bauer script.

bauer add bauer-plugin-fetch
bauer add bauer-plugin-scrape
bauer add bauer-plugin-extract

Runs the bauer script.

bauer run

Edit your bauer script making use of the plugins.

module.exports = function() {

  return this.Promise
    .fetch("http://http-bin.org")
    .print("Page downloaded.")
    .scrape({
      "a[href]": {
        links: {
          url: "attr:href",
          name: "text"
        }
      }
    })
    .extract("$..url")
    .print("Links matched.")
    .map(function(url) {
      return this.Promise.fetch(url);
    })
    .print("Links downloaded.")
    .repeat();
};
